11.2.1. Connect and detect the device¶

  1. Plug in the motherboard with the attached device on the computer and wait for the new device found windows wizard to complete.

  2. Open SmartSnippets™ Toolbox and select from the menu Board -> Detect Device

    Figure 179 Board -> Detect Device menu item

  3. Wait for the detection process to compete.

    Figure 180 Detection process

  4. Check Log window for any actions asked from the user (for example user may be asked to press the RESET button on the motherboard).

    Figure 181 Log window action

  5. When the process is finished a window like the following appears:

    Figure 182 Detection results

  6. Press OK. The menu items Board->UART, Board->Power Profiler and Board->JTAG are updated with ports and jtag serial numbers found on the attached device.
